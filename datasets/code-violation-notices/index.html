Streets Department, City of Philadelphia

Code Violation Notices

A code violation notice is issued when a person has violated one or more codes in the City of Philadelphia or violated one or more Streets Department rules and regulations. A code violation notice (CVN) is a penalty punishable by a fine up to $300.00. Addresses have been generalized to the hundred-block level (ie. 1234 Market Street becomes 1200 block of Market Street).
