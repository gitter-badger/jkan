Streets Department, City of Philadelphia

Bike Network - Supporting Datasets

This dataset combines data from various City of Philadelphia departments in order to create a set of supporting documentation for bike network routing. Datasets include Bike Network, Connector Streets, Regional Routes, and Trails and Side paths. Each dataset has its own parameters, dates, and sources, and individuals are encouraged to view the metadata associated with this data.

This dataset is experimental and remains a work in progress. Please use with caution.
